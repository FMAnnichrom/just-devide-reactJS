# Just Divide: Cat Puzzle

A lightweight, fully responsive arithmetic and logic puzzle game built entirely in a single HTML file with ReactJS and Tailwind CSS. 

## 🎮 How to Play
1. **Choose Your Difficulty:** Start by selecting Easy, Medium, or Hard. The game populates the board and generates upcoming tiles dynamically based on your chosen tier.
2. **Divide and Conquer:** Drag logic tiles from your active queue into the grid. If you place a tile adjacent to a cell where a division resolves mathematically (or identical tiles merge), they will combo! 
3. **Save for Later:** Use the **KEEP** slot to hold a tile out of rotation.
4. **Take out the Trash:** Use the **TRASH** bin to manually destroy tiles you can't use (limited uses per level).
5. **Chain Combos:** Connect math chains for huge combo multiplier points!

## 🛠️ Technology Stack
- **React 18** (No-build standalone via Babel)
- **Tailwind CSS** (Via CDN)
- **Vanilla JavaScript** (ES6 components)

## 🚀 Running the Game
This project possesses zero backend or pre-compilation dependencies. 
To play, just open `index.html` natively in any modern web browser or serve it via a basic static local server!

*Enjoy the math.*
